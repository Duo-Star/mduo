## [Windows 版本历史](https://zh.wikipedia.org/wiki/Microsoft_Windows%E7%9A%84%E5%8E%86%E5%8F%B2)

1. Windows 1.0 于1985年推出，原名 Interface Manager，觉得不够带[派](https://en.wikipedia.org/wiki/Pi)，于是改名Windows
2. Windows 2.0(Windows 2.1x) 1987年发行，比起上一版本较受欢迎
3. Windows 3.0(Windows 3.1x) 大受欢迎，两年内卖出一千万套，微软大赚一笔
4. Windows 95 / Windows 98 / Windows Me 分别于 95/98/00 年发布

## 古老的[Dos](https://zh.wikipedia.org/wiki/DOS)
Dos，磁盘操作系统（Disk Operating System）的缩写，是 20 世纪 80 年代到 90 年代中期个人电脑上最主流的操作系统。它是一种基于命令行界面（CLI）的单用户、单任务操作系统，用户通过输入特定的文本指令来控制计算机
